# 🍭 Jelly Shooter - Powered by Initia

Revolutionizing casual gaming by bringing back "Classic Hype Games" into the Web3 world, seamlessly integrated via the **Initia Interwoven Stack**.

## 🚀 Live Demo
[Play Jelly Shooter on Vercel](https://jelly-shooter-connect-initia.vercel.app/)

## 🎮 How to Play
- **Primary Shoot**: Click to blast the jellies!
- **🚀 Boost Nudge**: Press `SHIFT` or `N` to gain extra momentum.
- **🫨 Shake Attack**: Use the Shake feature for a tactical shooting advantage.
- **Goal**: Score 2100+ points to trigger the **On-Chain Masterclass Rewards**!

## ✨ Key Features
- **Nostalgic Revival**: Bringing classic arcade mechanics to Initia Layer 2.
- **Interwoven UX**: Seamless wallet synchronization using `InterwovenKit`.
- **Dual-Address Fix**: Optimized logic to bridge Keplr and Initia addresses (`init12` vs `init15` anomaly).
- **On-Chain Claims**: Automated reward system linked via Initiation-2 Testnet.

## 🛠️ Tech Stack
- **Frontend**: React.js, Vite, Vanilla CSS.
- **Blockchain**: Initia Layer 2 (Testnet).
- **Integration**: `@initia/interwovenkit-react`.
