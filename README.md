# 🍭 Jelly Shooter - Powered by Initia

Revolutionizing casual gaming with transparent on-chain reward mechanisms on the **Initia Network**.

## 🚀 Live Demo
[Play Jelly Shooter on Vercel](https://jelly-shooter-connect-initia.vercel.app/)

## ✨ Key Features
- **Interwoven UX**: Seamless wallet synchronization using `InterwovenKit`.
- **Dual-Address Fix**: Optimized logic to synchronize Keplr and Initia addresses (`init12` vs `init15` anomaly).
- **On-Chain Rewards**: Automated claim system linked via Initia Testnet.
- **Colorful UI**: A candy-themed arcade experience designed for mass adoption.

## 🛠️ Tech Stack
- **Frontend**: React.js, Vite, Vanilla CSS.
- **Blockchain**: Initia Layer 2 (Testnet).
- **Integration**: `@initia/interwovenkit-react`.
- **Deployment**: Vercel.

## 🔧 Installation
1. Clone the repo
2. `npm install`
3. `npm run dev`
